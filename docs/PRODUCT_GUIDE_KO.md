# 비개발자용 운영 가이드

## 지금 할 일 (PM/디자이너)
1. `npm run dev:api` 실행
2. 다른 터미널에서 `npm run dev:web` 실행
3. 웹(`http://localhost:3000`) 접속
4. 가입 후 미션 수행 → 피드백/점수 확인
5. 광고 정책 버튼으로 동의 전후 모드 확인
6. KPI 섹션에서 수치 변화를 확인

## 실험 루프 (주간)
1. 월요일: KPI 중 1개만 문제로 선정
2. 수요일: 해당 KPI를 올릴 UX/콘텐츠 실험 1개 배포
3. 금요일: 유지/수정/폐기 결정

## 권장 우선순위
1. 첫 미션 완료까지 5분 이내 달성
2. 미션 정답률 75% 근처로 난이도 조정
3. D7 유지율을 높이는 리마인더/보상 개선

## 운영 체크리스트
- 정책: 국가별 광고 규제 승인 상태 반영 (`AD_POLICY_APPROVAL_KR`, `AD_POLICY_APPROVAL_US`)
- 콘텐츠: 오답 피드백이 이해 가능한 문장인지 점검
- 데이터: 이벤트 누락 없이 퍼널 계산 가능한지 확인
